# 🚗 Parking System in Java

A Java-based Parking System application that simulates vehicle entry, exit, and parking slot allocation using object-oriented principles.

---

## 📖 Overview

This project implements a parking management system that tracks available parking slots and vehicle information. Users can add vehicles when they enter, remove them when they leave, and view current parking status. It demonstrates core Java concepts such as classes, objects, collections, and control flow.

---

## 🧠 Concepts Covered

- Object-Oriented Programming (OOP)
- Class and Object Design
- Collections (ArrayList, HashMap)
- Input/Output Handling
- Parking logic and slot management
